# Badge
---

添加 `.zw-badge` class 到 `<div>` 或者 `<span>` 元素。

## 默认样式

`````html
<span class="zw-badge">1</span>
<span class="zw-badge zw-badge-primary">2</span>
<span class="zw-badge zw-badge-secondary">3</span>
<span class="zw-badge zw-badge-success">4</span>
<span class="zw-badge zw-badge-warning">5</span>
<span class="zw-badge zw-badge-danger">6</span>
<span class="zw-badge zw-badge-success">Allmobilize</span>
<br>
<a class="zw-badge zw-badge-primary">Free</a>
<a class="zw-badge zw-badge-secondary">Secondary</a>
<a class="zw-badge zw-badge-success">Success</a>
<a class="zw-badge zw-badge-warning">Warning</a>
<a class="zw-badge zw-badge-danger">Danger</a>
`````

## 大小

结合[辅助类](/css/utility?_ver=2.x)中的字号 class，改变徽章大小。

`````html
<span class="zw-badge zw-badge-primary">default</span>
<span class="zw-badge zw-badge-secondary zw-text-sm">small</span>
<span class="zw-badge zw-badge-success zw-text-default">normal</span>
<span class="zw-badge zw-badge-warning zw-text-lg">large</span>
<span class="zw-badge zw-badge-danger zw-text-xl">x large</span>
`````
